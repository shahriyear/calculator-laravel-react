# Calculator  
  
Emoji Based Calculator Laravel and React.  

### Versions

PHP 7.3.33
NODE 12.22.10
NPM 6.14.16
LARAVEL 8.75


### Installation
  
Clone repository and install dependencies via composer.  
  
```php

➜  cd calculator

➜  composer install

➜  npm install

➜  npm run dev

➜  php artisan serve

```
